命令行搜索引擎

格式：
> ./init [OPTINES] 
> ./Query

`` ./init ``

创建字典和倒排表

`` ./init --clean ``

删除所创建的字典和倒排表

`` ./init --help ``

帮助

`` ./Query ``

检索信息

`` /Query --help ``

帮助
